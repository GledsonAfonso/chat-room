# chat-room

> A simple chat-room project to study web sockets.

## Prerequisites

- Java 21
- Maven 3.6.3

## Setting up the server

First, open a terminal in the `server` dir. Compile it with:

```bash
mvn clean install
```

And then, run it with:

```bash
java -jar ./target/chat-room-server-1.0.jar
```
